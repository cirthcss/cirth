# Colors

Cirth's palette has 20 accent colors, each with hand-tuned light and dark
variants. One color is picked at **build time** — it becomes `--cirth-primary`
and the rest of the primary color group — so switching accents means loading
a different published stylesheet, not passing a runtime flag.

<ColorSwatches />

`azure` is the default (`dist/cirth.min.css`, no color suffix). Every other
color is published as a named build, for both the default and classless CSS,
scoped and unscoped:

```html
<!-- Default build, jade accent -->
<link rel="stylesheet" href="dist/cirth.jade.min.css">

<!-- Classless build, slate accent -->
<link rel="stylesheet" href="dist/cirth.classless.slate.min.css">

<!-- Scoped build, pumpkin accent -->
<link rel="stylesheet" href="dist/cirth.scoped.pumpkin.min.css">
```

The full color-build matrix is generated by `scripts/build-themes.js` during
`npm run build`; see [Build tooling](/build-tooling) for how it works.

## Picking a different primary without a new build

If you only need a one-off accent rather than a full theme swap, override the
primary color group with your own values instead of loading a different
build — see [Customization](/customization#primary-secondary-and-contrast).
This is the right tool when a single page or component needs a custom brand
color; use a named color build when you want the whole site to consistently
use one of Cirth's tuned accents (with its matching hover/focus/dark
variants already balanced for contrast).

## Underlying palette

Every accent is built from a 19-step scale (`50` lightest to `950` darkest)
defined in `src/colors/_index.scss`. These Sass variables aren't part of the
public CSS custom property surface — they only exist at build time to derive
the semantic tokens (`--cirth-primary`, `--cirth-primary-hover`, …) baked
into each color's stylesheet.

## Roadmap note

The current 20-color set is inherited from Pico CSS and is under review —
see the [project roadmap](/about#roadmap). Expect the published set to
shrink toward a smaller, more curated group of maintained colors.
