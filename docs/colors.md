# Colors

Cirth ships three themes, each with hand-tuned light and dark variants. A
theme is picked at **build time** — switching themes means loading a
different published stylesheet, not passing a runtime flag.

<ColorSwatches />

`azure` is the default (`dist/cirth.min.css`, no theme suffix). Jade and
Slate are published as named builds, for both the default and classless
CSS, scoped and unscoped:

```html
<!-- Default build, jade theme -->
<link rel="stylesheet" href="dist/cirth.jade.min.css">

<!-- Classless build, slate theme -->
<link rel="stylesheet" href="dist/cirth.classless.slate.min.css">

<!-- Scoped build, jade theme -->
<link rel="stylesheet" href="dist/cirth.scoped.jade.min.css">
```

The full theme-build matrix is generated by `scripts/build-themes.js` during
`npm run build`; see [Build tooling](/build-tooling) for how it works.

## Picking a different primary without a new build

If you only need a one-off accent rather than a full theme swap, override the
primary color group with your own values instead of loading a different
build — see [Customization](/customization#primary-secondary-and-contrast).
This is the right tool when a single page or component needs a custom brand
color; use a named theme build when you want the whole site to consistently
use one of Cirth's tuned themes (with its matching hover/focus/dark variants
already balanced for contrast).

## Underlying palette

Every theme is built from 19-step color scales (`50` lightest to `950`
darkest) defined in `src/colors/_index.scss`. These Sass variables aren't
part of the public CSS custom property surface — they only exist at build
time to derive the semantic tokens (`--cirth-primary`,
`--cirth-primary-hover`, …) baked into each theme's stylesheet.

Jade and Slate are complete, self-contained themes under `src/themes/jade`
and `src/themes/slate` — not accent swaps of the default theme. Only the
default theme keeps a color-mapping layer (`src/themes/default/_theme-colors.scss`),
and it now holds a single entry, `azure`.

## Roadmap note

Cirth previously inherited a 20-color accent set from Pico CSS. That set has
been reduced to three maintained themes — azure (default), jade, and slate —
see the [project roadmap](/about#roadmap).
